# MS-API

## 開発準備

```
# GO111MODULEをオンにする
$ export GO111MODULE=on

# App初期設定
$ make init
```

## 実行コマンド

```
# 実行
$ make run

# Dockerビルド
$ make docker-build

# Docker実行
$ make docker-run

# external appsを立ち上げる
$ make external-run

# external appsを終了する
$ make external-end

# ヘルプ
$ make help
```
